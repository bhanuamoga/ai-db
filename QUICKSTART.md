# Quick Start Guide

Get QueryAI running in 5 minutes.

## 1. Install Dependencies

```bash
npm install
# or
pnpm install
```

## 2. Run Development Server

```bash
npm run dev
# or
pnpm dev
```

## 3. Open in Browser

Visit [http://localhost:3000](http://localhost:3000)

## 4. Login with Demo Account

- **Email**: demo@queryai.com
- **Password**: demo123

## 5. Explore Features

- **Dashboard**: AI chat interface for database queries
- **Connections**: Manage database connections
- **History**: View past queries and results
- **Settings**: Customize theme and preferences
- **Billing**: View pricing plans

## Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel

# Or use the dashboard at vercel.com
```

That's it! See [DEPLOYMENT.md](./DEPLOYMENT.md) for detailed deployment instructions.
